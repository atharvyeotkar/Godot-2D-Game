# 🎮 Godot 2D Platformer Game

A simple 2D platformer game developed using **Godot Engine** and **GDScript**. The game includes player movement, enemies, coins, platforms, kill zones, and background music.

## ✨ Features

- 🎮 Smooth player movement
- 🪙 Coin collection system
- 👾 Enemy collision
- ☠️ Kill Zone (Player Respawn)
- 🏃 Platform-based gameplay
- 🎵 Background music
- 🕹️ Built using Godot Engine

---

## 🛠️ Built With

- Godot Engine 4.x
- GDScript

---

## 🎮 Controls

| Key | Action |
|------|--------|
| ⬅️ Left Arrow / A | Move Left |
| ➡️ Right Arrow / D | Move Right |
| ⬆️ Up Arrow / Space | Jump |

## 🚀 How to Run

1. Install **Godot Engine 4.x**
2. Clone this repository

```bash
git clone https://github.com/atharvyeotkar/Godot-2D-Game.git
```

3. Open the project using Godot.
4. Run the main scene.
